# C# Basics Training

Этот репозиторий содержит мои учебные упражнения по языку C#.

## Темы, которые отрабатывались:
- базовый синтаксис C#
- условия и циклы
- массивы и коллекции
- методы
- основы ООП

## Цель
Закрепить базовые концепции языка C# на практике.

## Структура
Каждая папка содержит отдельные задания и решения с комментариями.
